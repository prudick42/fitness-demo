## Шаг 1: Установка зависимостей 

Выполните установку необходимых пакетов Python:

```bash
pip install -r requirements.txt
```

## Шаг 2: Установка базы данных

Выполните создание пустой базы данных:

```bash
psql -U postgres -c "CREATE DATABASE gym_db;"
```

Восстановите бэкап:

```bash
# Выполнить из папки проекта
psql -U postgres -d gym_db -f gym.sql
```

## Шаг 3: Запуск приложения

Выполните команду:

```bash
python app.py
```
Перейдите по локальной ссылке из терминала:

```bash
http://127.0.0.1:5000
```
P.S. если появляется ошибка проверьте совпадение портов бд в коде(5433->5432)
